Alur Pelaporan




Pertama

User yang terkena penipuan dan ingin melaporkan terlebih dahulu melakukan login ke Website Laporin apabila belum mempunyai akun maka diharuskan untuk mendaftar.

Kedua

User diminta untuk menjelaskan kronologi penipuan dan mengisi data-data yang ada. bukti yang dikirmkan haruslah valid seperti screenshoot transfer ke rekening penipu dll.

Ketiga

Setelah user menginput semua data maka laporan akan masuk ke waiting list admin. pada tahap ini user masih bisa menghapus laporan

Keempat

Admin yang sedang bertugas akan melakukan pengecekan hasil dari user apabila bukti yang dikirimkan belum kuat maka admin akan menolak laporan.

Kelima

Apabila laporan diterima oleh admin maka laporan akan publish dan nomor rekening atau nomor e-wallet pelaku akan tertanda.

Keenam

Apabila user dan penipu telah berdamai dan uang telah dikembalikan maka user diperkenankan untuk menghapus laporan atau membiarkannya agar user lain yang ingin mengecek nomor tersebut bisa tahu.
